# Universal Hub

A versatile Roblox script that works in every game, with expanded feature sets for select popular ones. Lightweight, transparent, and updated often.

**→ [angxers2.github.io/Unihub](https://angxers2.github.io/Unihub)** — supported games, showcase, changelog.

## Loadstring

```lua
loadstring(game:HttpGet("https://raw.githubusercontent.com/Angxers2/Unihub/main/Unihub%20V3.0.lua"))()
```
