# Universal Hub Lite

A lightweight, chat-based build of Universal Hub. Smaller footprint, faster boot, no GUI — control everything via `!commands` in chat.

**→ [angxers2.github.io/Unihub](https://angxers2.github.io/Unihub)** — full command list, showcase, changelog.

## Loadstring

```lua
loadstring(game:HttpGet("https://raw.githubusercontent.com/Angxers2/Unihub/lite/Unihub%20Lite%20V1.2"))()
```
